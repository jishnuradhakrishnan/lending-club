tech stack used:-
java8,agular7, spring boot,maven, npm, mysql

to run this app(in terminal or in cmd):-
java -jar LendingClub-0.0.1-SNAPSHOT.jar

to run this app using your mysql credentials(change my parameter values with yours):-
java -jar LendingClub-0.0.1-SNAPSHOT.jar --spring.datasource.url=jdbc:mysql://localhost:3306/LENDING_CLUB?createDatabaseIfNotExist=true --spring.datasource.username=root --spring.datasource.password=12345
